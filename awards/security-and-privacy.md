# Security and privacy (public-safe summary)

## Public delivery controls

- PR-only delivery and CODEOWNERS where configured
- CI, CodeQL, Dependabot, and Scorecard patterns on flagship repos
- Public Proof Gate for claims and privacy-boundary checks
- Secrets and private runtime evidence excluded from public repositories

## Privacy posture for awards

Submissions use only public repositories, public documentation, and sanitized architecture summaries. No credentials, client data, employer-confidential procedures, or private system internals are included.

## Analytics and intake

Portfolio analytics and newsletter collection are deferred until privacy review is complete. GitHub Pages remains a technical showcase, not a sensitive intake system.
