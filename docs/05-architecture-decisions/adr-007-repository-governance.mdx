# ADR-007: Repository Governance

## Status

**Accepted** — July 2026

## Context

Public engineering credibility requires delivery controls, not only attractive READMEs.

## Decision

Flagship public repositories and the BBI organization baseline should include, as applicable:

1. PR-only protected defaults
2. CODEOWNERS
3. CI, CodeQL, Dependabot, Scorecard patterns
4. Public Proof Gate / privacy scans for claim surfaces
5. Honest status labels (Released / Active / Planned)

## Consequences

Higher setup cost; stronger recruiter and awards signal; fewer accidental publishes.
