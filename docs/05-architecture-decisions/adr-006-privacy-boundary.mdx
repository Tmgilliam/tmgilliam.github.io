# ADR-006: Privacy Boundary

## Status

**Accepted** — July 2026

## Context

BBIOS and related private systems inform methodology but must not leak into public repositories, awards packs, or portfolio pages.

## Decision

1. Public content is sanitize-or-omit
2. Private systems may appear only as labeled architecture summaries
3. No Nova/MLS/Myelin internals, client data, or credentials in public artifacts
4. Awards evidence packs inherit the same boundary

Supplements [ADR-002](./adr-002-public-proof-boundary).

## Consequences

Less “full access” theater for recruiters; more durable trust and legal/safety posture.
