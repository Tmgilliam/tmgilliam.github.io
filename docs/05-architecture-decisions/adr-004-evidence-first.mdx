# ADR-004: Evidence First

## Status

**Accepted** — July 2026

## Context

Portfolios and AI products often assert readiness or impact without artifacts a stranger can verify. That destroys recruiter and judge trust.

## Decision

Adopt **evidence first** as a portfolio and product rule:

1. Prefer releases, CI, docs, ADRs, and demos over narrative claims
2. Label assumptions and limitations explicitly
3. Never present static fixtures as live telemetry
4. Score award submissions with a Matrix ≥85 before entry

## Consequences

Slower storytelling; higher credibility. Some impressive private work remains unpublished by design.
