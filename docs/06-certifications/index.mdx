# Certifications

Formal credentials do not replace working code—but they establish a **reviewed baseline** of platform knowledge that recruiters and architects can trust when paired with public repository evidence.

## Completed credentials (Azure)

| Exam   | Certification                              | Role path layer |
| ------ | ------------------------------------------ | --------------- |
| AZ-900 | Microsoft Azure Fundamentals               | Foundation      |
| AZ-104 | Microsoft Azure Administrator              | Operations      |
| AZ-305 | Microsoft Azure Solutions Architect Expert | Architecture    |

See [Azure Solutions Architect Expert](../certifications/azure-architect) for the capstone credential and how it maps to portfolio work.

## How certifications appear in this portfolio

- **Cloud Architecture** sections reference AZ-305 design pillars (reliability, security, cost, operations, performance)
- **Public repos** provide implementation evidence—not every exam topic has a dedicated public artifact
- **No certification** is presented as a guarantee of client outcomes or production deployments

## Verification

Microsoft credentials can be verified through [Microsoft Learn transcript / Credly badges](https://learn.microsoft.com/credentials/) when the holder shares public badge links. This site describes completed exams based on the holder's recorded achievement; add badge URLs when published for third-party verification.
