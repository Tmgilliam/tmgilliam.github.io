# Living Engineering Platform Roadmap

Elite Docusaurus sites earn trust through **information architecture, polished execution, and immediate proof**—not animated backgrounds or vanity counters.

## Assessment (working scores)

| Area                       | Current | Elite target |
| -------------------------- | ------: | -----------: |
| Visual design              |     8.8 |          9.8 |
| Engineering credibility    |     8.5 |           10 |
| Recruiter experience       |       8 |           10 |
| Storytelling               |     8.5 |           10 |
| Interactive experience     |       6 |          9.5 |
| Technical authority        |       8 |           10 |
| Open-source attractiveness |       7 |          9.5 |

Biggest opportunity: make the site feel like a **living engineering platform**.

## Phase 1 — Recruiter excellence (now)

```text
Executive homepage with proof CTAs
Recent Shipments
Curated projects catalog
Architecture hub
Engineering Mission Control
Ecosystem relationship map
Architecture explorer (v1)
Expanded ADRs
Engineering principles
Case-study structure on flagship docs
```

## Phase 2 — Engineering excellence

```text
Deeper interactive architecture explorer
Public quality gates panel (live where possible)
Search everywhere (Algolia or local index — keys required)
Versioned docs where product surfaces need it
Live roadmap embeds / tighter project sync
```

## Phase 3 — Community and authority

```text
Technical articles cadence
Engineering journal / Sunday review
Conference talks and workshops
Open-source contribution tracker
Recognition page populated only with verified achievements
Executive Mode / Developer Mode toggle
```

## Explicitly deferred

```text
Animated backgrounds
Vanity contribution counters
Embedded contribution graphs
Newsletter / waitlist until privacy review
Client/employer logos without permission
Live private BBIOS tracking
```

## Differentiator to protect

**Evidence-first enterprise AI** — systems where consequential actions are authorized, traceable, and supported by evidence.
